# 打开蓝屏文件收集

此文档用于打开电脑蓝屏文件收集功能 当发生蓝屏问题 系统未处于完全不能进入的情况下 取出蓝屏文档有助于分析蓝屏问题的产生 

按此文档产生的蓝屏文件路径C:\Windows\Minidump  

请在生成后发给技术人员分析 谢谢配合

右键此电脑选择 属性
